Vamos, depois de iniciar o ambiente lendo agora a base de dados tratada, voltando a utilizar o value_counts() para passar por todas as colunas e ver com mais calma o que cada uma têm.

import polars as pl
df = pl.read_csv('base_fapesp_tratada.csv')

# Para cada coluna existente na lista de colunas de df
for col in df.columns:
    # Imprima a contagem de valores desta coluna, em ordem decrescente    
    print(df[col].value_counts(sort=True))

    # E uma linha em branco, para fiar mais organizado.
    print()

  1. Quais são as cidades que mais recebem financiamento? - col relevante: Cidade Instituição

  2. Quais são as principais áreas do conhecimento e como elas se dividem entre as grandes áreas? - cols relevantes: Grande Área do Conhecimento e Área do Conhecimento

  3. Quais são as instituições que mais recebem financiamento? - col relevante: Instituição

  4. Como se deu a distribuição das bolsas no tempo? - cols relevantes: Data de Início e Data de Término
